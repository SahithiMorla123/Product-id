# Product ID Management

A REST API built with Java and Spring Boot to retrieve and manage product IDs from the system.

## Features
- Fetch all product IDs
- Retrieve product by specific ID
- Validate product data accuracy

## Technologies Used
- Java
- Spring Boot
- REST API
- Eclipse IDE
- Postman (for API testing)

## How to Run
1. Clone the repository
2. Open in Eclipse IDE
3. Run as Spring Boot Application
4. Test endpoints using Postman
